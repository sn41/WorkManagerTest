## Как проверять эту работу (Сценарий для студента):

**Запуск**: Запустите приложение на эмуляторе.

**Действие**: Введите любой URL и нажмите кнопку.

**Наблюдение 1 (Running)**: Сразу после нажатия статус должен смениться на "В очереди", а затем (почти мгновенно) на "Выполняется". Должен появиться прогресс-бар.

**Сворачивание**: Пока крутится прогресс (у вас есть 3 секунды), быстро сверните приложение (нажмите Home).

**Наблюдение 2 (Logcat)**: Откройте Logcat в Android Studio. Вы должны увидеть сообщения: Начинаю загрузку... и затем Качаем данные..., даже когда приложение свернуто.

**Возврат**: Разверните приложение обратно. Статус должен быть "Готово!" (зеленый), и вы должны увидеть текст результата с ID.

Это доказывает, что WorkManager выполнил задачу в фоновом потоке, и она пережила сворачивание приложения.
